# 💰 Command Line Banking System

A simple command-line based banking system built using Python. It allows users to create accounts, deposit and withdraw money, and check their balance—all from the terminal.

## 📁 Project Structure


## 🚀 Features

- Create new user accounts with unique IDs  
- Deposit funds into an account  
- Withdraw funds with balance checks  
- Check account balance  
- Calculate interest (Savings accounts only)  
- Command-line interface for interaction

## 🧠 How It Works

- `account.py`: Defines the `BankAccount` class and its subclasses `SavingsAccount` and `CurrentAccount`.
- `transactions.py`: Contains logic for deposit and withdrawal operations.
- `main.py`: Provides the CLI interface for users to interact with the system.
